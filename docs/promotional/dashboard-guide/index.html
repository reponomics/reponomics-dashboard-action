Lead Story and Next Moves

The focused article creates the first read; the queue preserves the practical next-move list for scanning and repo focus.

What to look for

Purpose

Create an above-the-fold story without inventing analysis: the lead card reframes existing rules-based prompts as a focused read.

Carousel

Story tabs and arrow controls rotate through the strongest prompts for the selected window.

Affordance

Lead buttons and queue cards focus the associated repo across the dashboard.

Use case

Useful for flat or mixed windows because it emphasizes what to do next, not only what already spiked.

Opportunity Map

Clustered demo data now uses smaller marks so labels and quadrants stay readable.

What to look for

Axes

Right means more attention; up means more downstream growth from stars, forks, and subscribers.

Bubble size

Size follows clone activity. The scale is compact so crowded portfolios still read cleanly.

Quadrants

Labels suggest practical recipes such as clarify next step or amplify.

Affordance

Points and right-side notes focus the repo. Hover/title text carries exact values.

Code Activity Ribbon

A low-noise event layer for spotting traffic-adjacent codebase activity.

What to look for

Purpose

Bring default-branch commits and releases into the same time window as traffic, so maintainers can inspect temporal adjacency.

Encoding

Daily clusters aggregate commits and releases; releases use the stronger diamond marker.

Nearby views

Tooltips and log items show nearby traffic counts as context for follow-up.

Affordance

Single-repo clusters and log rows focus the repo inside the dashboard; multi-repo clusters stay as contextual hover targets.

Readiness Queue

A compact path from current attention to public-facing setup improvements.

What to look for

Purpose

Convert attention into practical maintenance actions: README, license, contributing guide, templates, and conduct files.

Score

The score uses known community-health checks and reports visible coverage.

Priority

Repos with setup gaps and current views plus clones rise in the queue.

Affordance

Fix cards focus the repo and make the next maintenance step explicit.

Tables and Source Inspection

Use the tables when you need the underlying rows behind a visual or prompt.

What to look for

Referrers

Shows where attention came from in the selected window; column headers sort the table.

Popular paths

Useful for spotting docs, release notes, or entry pages that deserve a clearer next step.

Repositories

Detailed per-repo metrics backstop the heuristic sections with raw values.

Mobile behavior

Wide tables scroll inside the table area so the page layout remains stable.
