# Security Guidance

This dashboard repository is owned by its repository owner. Reponomics does not administer user-created dashboard repositories, repository secrets, workflow permissions, Pages settings, branch protections, or collaborator access.

For dashboard-key handling, see [Secure Dashboard Key Generation](secure-dashboard-key.md). For repository access and collaborator trust boundaries, see [Repository Access And Trust Boundary](trust-boundary.md). For action release and supply-chain verification, see [Provenance And Supply Chain Verification](provenance.md).

Do not publish secrets, dashboard keys, retained artifact contents, or exploit details in public issues. If this repository belongs to someone else, follow that repository owner's reporting process. If you are reviewing Reponomics itself, use the current upstream Reponomics security reporting guidance.
